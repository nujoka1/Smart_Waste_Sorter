# IoT-Based Smart Waste Sorting System

Target controller: ESP32 WROOM-32
Upload method: Arduino IDE
Main firmware: firmware/SmartWasteSorter/SmartWasteSorter.ino

Core bins:
1. Plastic
2. Metal
3. Wet / Organic
4. Paper / Glass
5. Unknown

No weight sensor. Bin monitoring uses ultrasonic fill-level sensing.
